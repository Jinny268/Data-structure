<AVL트리>

1. 균형인수
 - 노드 X의 균형인수는 BF(X)라고 한다.
 - hL은 X의 왼쪽 서브트리의 높이이다.
 - hR은 X의 오른쪽 서브트리의 높이이다.
 - BF(X)=hL-hR

2. AVL트리
 - 모든 노드 X에 대해서 BF(X)=-1, 0, 1이다.
 - AVL트리의 높이는 O(logn)이다.

3. 피보나치 수
 - Fib(n): n번째 피보나치 수
   Fib(n) = 0     if    n = 0
              1     if    n = 1
              Fib(n-1) + Fib(n-2) if n > 1
 - F(h) = Fib(h+2)

4. AVL트리의 원소 삽입
 (1) Y: 이진탐색트리에서 삽입된 노드
 (2) 삽입된 노드 Y에 가장 가까우면서 균형인수가 ±2인 조상 노드 A에 
     대하여, 다음 4가지 경우로 나누어 회전
     - LL: 새 노드 Y는 A의 왼쪽 서브트리의 왼쪽 서브트리에 삽입
     - LR: Y는 A의 왼쪽 서브트리의 오른쪽 서브트리에 삽입
     - RR: Y는 A의 오른쪽 서브트리의 오른쪽 서브트리에 삽입
     - RL: Y는 A의 오른쪽 서브트리의 왼쪽 서비트리에 삽입
 (3) 단일 회전: LL과 RR의 경우 불균형을 바로잡는 변환
 (4) 이중 회전: : LR과 RL의 경우 불균형을 바로잡는 변환

5. 회전 방법
 (1) LL: A부터 Y까지의 경로상의 노드들을 오른쪽으로 회전시킨다. 
 (2) LR: A부터 Y까지의 경로상의 노드들을 왼쪽-오른쪽으로 회전시킨다. 
 (3) RR: A부터 Y까지의 경로상의 노드들을 왼쪽으로 회전시킨다. 
 (4) RL: A부터 Y까지의 경로상의 노드들을 오른쪽-왼쪽으로 회전시킨다. 

6. AVL트리의 수행시간 (n: 노드 수)
 (1) 원소 검색: O(logn)
 (2) 원소 삽입: O(logn)
 (3) 원소 삭제: O(logn)
